# ad-user-automation-powershell
🔐 Active Directory User Automation with PowerShell

📌 Overview

This project demonstrates how to automate user account creation in Active Directory using PowerShell within a virtual machine environment.

The goal of this lab is to streamline user provisioning by eliminating manual steps and leveraging scripting for efficiency and scalability.

	•	Windows Server (Virtual Machine)
	•	Hypervisor (VirtualBox / VMware / etc.)

⸻

⚙️ How It Works
	1.	Create a new PowerShell script file
	2.	Save the file with a .ps1 extension
	3.	Open PowerShell as an administrator
	4.	Execute the script to create a new user
	5.	Verify the user account in Active Directory
